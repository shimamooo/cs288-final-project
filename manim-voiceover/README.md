# manim-voiceover

[pypi page](https://pypi.org/project/manim-voiceover/)

Follow [manim-voiceover installation guide](https://voiceover.manim.community/en/stable/installation.html) but just replace `manim-voiceover` for `manim-voiceover`. For example:

```
pip install --upgrade "manim-voiceover[azure,elevenlabs]"
```

This is a fork from https://github.com/ManimCommunity/manim-voiceover/.

Unlike the upstream repository, this repository fully supports [elevenlabs@2.1.0](https://pypi.org/project/elevenlabs/2.1.0/).

View [manim_voiceover/services/elevenlabs.py](./manim_voiceover/services/elevenlabs.py) for all available options, and [examples/elevenlabs-example.py](./examples/elevenlabs-example.py) for an example.
